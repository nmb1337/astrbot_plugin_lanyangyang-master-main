# astrbot_plugin_lanyangyang

懒羊羊主题基础群管插件。插件自身的命令回复会生成羊羊风格图片卡片，卡片里会显示呼叫人的头像和名称；开启文本装饰后，只装饰 AstrBot 的 AI 纯文本回复，不会改写其他插件的消息格式。

## 功能

- 命令支持直接发送或带 `/` 发送，例如 `菜单`、`/菜单`
- 发言统计：`发言统计`、`我的统计`
- 邀请排行：`邀请排行`（依赖协议端上报邀请人）
- 群管：`禁言 @成员 10m`、`解禁 @成员`、`撤回`、`批量撤回 5`、`踢出群 @成员`、`踢黑 @成员`、`上管 @成员`、`下管 @成员`
- 白名单：`拉白 @成员`、`删白 @成员`、`白名单`，白名单成员会跳过禁词、刷屏、宵禁自动处罚
- 娱乐：`点歌 qq 123456`、`点歌 网易 123456`、`语音`、`语音 文件名关键词`
- 语音/回怼：喊 `懒羊羊回家` 或艾特机器人时，会按概率回复
- AstrBot 的 AI 纯文本回复可自动装饰为懒羊羊主题图片，Zero、数据库等其他插件的回复保持原样

## 权限边界

- 只有本群群主、本群群管理员或 AstrBot 管理员可以执行：禁言/解禁他人、全体禁言、撤回/批量撤回、踢人/踢黑、修改他人群名片或头衔、维护白名单、设置群名、发布公告、设置禁词和刷屏处罚、发起禁言投票。
- `上管`、`下管` 只能由本群群主或 AstrBot 管理员执行。普通 QQ 群管理员不能借用群主机器人给别人授权或撤权。
- QQ 群管理员不能操作群主，也不能踢、禁言、改名或改头衔同级管理员；群主和 AstrBot 管理员仍受 QQ/NapCat 自身权限限制。
- 普通成员仍可使用只影响自己的 `禁我`、`改我`、`申请头衔`，也可查看统计/公告、参与已经由管理员发起的禁言投票和使用娱乐命令。
- 权限会实时通过 OneBot `get_group_member_info` 回查；无法确认身份时默认拒绝执行群管动作。

## 语音

把 `.wav`、`.mp3`、`.amr`、`.silk` 文件放到 `voices/` 目录。触发时插件会随机发送其中一个语音。

如果没有语音文件，插件会生成一张图片回怼。

## 说明

禁言、撤回、踢出、踢黑等操作依赖 QQ OneBot/aiocqhttp 的协议端 API，而且机器人账号本身必须拥有相应群权限。其他平台会正常显示图片提示，但无法执行这些平台专属群管动作。

图片主题使用羊羊风格原创卡片元素，不打包原作版权素材。
